# Portfolio - Mathis HAMRI

Bienvenue sur le dépôt de mon portfolio personnel. Il s'agit d'un site web statique qui présente mon profil, mes compétences, mes projets ainsi que mes coordonnées.

## 🚀 Fonctionnalités

- **Présentation :** Une introduction sur moi et mon parcours.
- **Projets :** Une liste détaillée de mes réalisations académiques et personnelles (Cybersécurité, Intelligence Artificielle, Développement Mobile, etc.).
- **CV :** Un accès direct pour télécharger mon Curriculum Vitae.
- **Contact :** Un formulaire fonctionnel (via Formspree) pour me contacter directement par e-mail, ainsi que mes liens vers GitHub et LinkedIn.
- **Thème Sombre / Clair :** Un bouton "toggle" pour passer d'un mode de couleur à l'autre de manière fluide.

## 🛠️ Technologies Utilisées

Ce projet a été réalisé de A à Z sans framework, en utilisant les standards du web :
- **HTML5** pour la structure.
- **CSS3** pour le design moderne, les animations, et la pleine compatibilité mobile (Responsive).
- **JavaScript (Vanilla)** pour l'interactivité (smooth scroll, changement de thème, validation de formulaire).

## 📂 Installation & Utilisation

Aucune installation n'est requise. C'est un site statique classique.

1. Clonez ou téléchargez ce dépôt.
2. Ouvrez simplement le fichier `index.html` dans le navigateur web de votre choix (Chrome, Firefox, Safari, Edge...).
3. Profitez de la visite !
